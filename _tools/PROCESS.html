Language pack creation process

Preliminaries

Set up fonts

  1. Create /shared/fontlists/scriptname.js
  2. Download needed fonts
  3. Make webfont(s) and add to /shared
  4. Set font in localstyle.css

Create a spreadsheet

Copy a similar script in and arrange columns

Get all characters from UniView block(s), rotate, and add.

Organise into consonants, vowels, marks, punctuation, etc.

Add phonetics from Wikipedia.

Add transliteration based on phonetics.

Add class info and additional entries based on Notes:instructions/Google Spreadsheet Guidelines.

Add unicode names using _list_codes_and_names.

Add block name column.

Create a character app

Get a template

Do case-sensitive change of script name throughout index.html

Set defaults

  1. Edit defaults.js
  2. Create sample text

Position keys

Copy characters to the appropriate place.

Or, if needed, create code using the following:

  1. Plan the zones for characters
  2. Use UniView to make a list of characters for the picker
  3. Use _make-characters-for-input.html to generate spans
  4. Add result to #temp div
  5. Move #temp markup to appropriate positions

Make shape list

  1. Use the picker to create a list of similar glyphs on each line, separated by spaces.
  2. Move the simplest shape to the start of the list.
  3. Create the code using _make-shape-list.
  4. Copy to main source inside <div id="shapekeys">.

Make hint list

  1. Take the list created in the last step.
  2. Create the code using hint-maker.
  3. Copy to localcode _h.

 

Create a character notes page

Copy a template for block & local.css.

Create initial code for all characters in block order using generate_block_page_content_NEW.

Organise into consonants, vowels, marks, punctuation, etc.

Add phonetics from Wikipedia.

Add transliteration based on phonetics.

Add class info and additional entries based on Notes:instructions/Google Spreadsheet Guidelines.

Add unicode names using _list_codes_and_names.

Add block name column.
