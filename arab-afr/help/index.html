This character app allows you to produce, analyse or manipulate runs of Unicode text in African languages using Arabic characters. Character apps are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes characters in various Unicode Arabic blocks that are used for writing languages of Africa. The original list was expanded to include research done by Neil Patel at Jamra-Patel.

The characters in the chart are arranged so that similar-shaped characters are together, to aid recognition. This, of course, only applies to the isolated forms, but when you mouse over the shapes, the character app displays the joining forms just above.
