This Unicode character picker allows you to produce or analyse runs of Armenian text. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes all characters in the Unicode Armenian block.

Controls above the text area

Transliterate. Converts the contents of the text area to a Latin transliteration that provides a one-to-one correspondence between Armenian and Latin symbols.

The transliteration scheme was developed for this picker, and, where possible, uses symbols that represent sounds typically associated with their Armenian equivalents, rather than the sometimes fairly arbitrary selection of diacritics found in other systems. Of course, the complexities of mapping graphemes to sounds means that this is far from a real phonetic transcription, but the process is reversible (unlike most transcriptions).

Armenian to ISO. Converts the contents of the text area to an ISO 9985 Latin transcription.

Input aids

ISO to Armenian. Allows you to generate Armenian text from an ISO 9985 transcription. It works as described in the section just above about Reverse transliteration, but uses a different panel to set up the transcription type-in mode.
