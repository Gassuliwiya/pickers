List of pickers

Updated 13-Dec-2018 • tags apps, pickers, bengali

This Unicode character picker allows you to produce or analyse runs of Bengali text. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes characters in the Unicode Bengali block.

Controls above the text area

Transliterate. Converts the contents of the text area to a Latin transliteration that provides a one-to-one correspondence between Bengali and Latin symbols.

The transliteration scheme was developed for this picker, and, where possible, uses symbols that represent sounds typically associated with their Bengali equivalents, rather than the sometimes fairly arbitrary selection of diacritics found in other systems. Of course, the complexities of mapping graphemes to sounds means that this is far from a real phonetic transcription, and it doesn't show inherent vowels, but the process is reversible (unlike most transcriptions).

Transliterations produced this way are particularly useful for understanding the composition of text in complex scripts, where the individual parts are not well separated in the text.

Bengali to ISO. Converts the contents of the text area to an ISO 15919 Latin transliteration.

This transliteration adds 'a' to indicate possible locations for the inherent vowel, however, this is not entirely predictable in Bangla. Consonants at the end of a word may or may not end with an inherent vowel, so you are offered a choice – click on the yellow background to select what you want in the final result.

ISO 15919 to Bengali. Converts text in an ISO 15919 transcription to Bengali characters.

Bengali to Radice. Converts Bengali text to the transcription used in Radice's book Teach Yourself Bengali.

Radice to Bengali. Converts text to Bengali characters from the transcription used in Radice's book Teach Yourself Bengali.

Input aids

ISO to Bengali. Allows you to generate Bengali text from an ISO 15919 transcription. It works as described in the section just above about Reverse transliteration, but uses a different panel to set up the transcription type-in mode.

Radice to Bengali. Displays a panel that allows you to generate Bengali text from a transcription as used by Radice's book Teach Yourself Bengali. It works as described in the section just above about Reverse transliteration, but uses a different panel to set up the transcription type-in mode.
