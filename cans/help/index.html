This Unicode character app allows you to produce or analyse runs of UCAS text. It is particularly useful for locating infrequently used characters.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes characters in the Unified Canadian Aboriginal Syllabics, and UCAS Extended blocks.

To find a character in this character app, click on one of the orange items in the selection area that resembles the shape of the character you are looking for. Then click on one of the grey shapes that appears below to add the character to the text area. The orange ellipsis (...) gives access to a mix of characters with shapes that didn't fit elsewhere.
