This Unicode character picker allows you to produce or analyse runs of Cyrillic text. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes most, but not all characters in the 5 Unicode Cyrillic blocks. It doesn't include combining characters, symbols or punctuation.

To add a character in this picker, click on one of the orange items in the selection area that ressembles the shape of the character you are looking for. Then click on one of the grey shapes that appears below to add the character to the text area. The orange ellipsis (...) gives access to a mix of characters with shapes that didn't fit elsewhere.

Controls above the text area

Uppercase. Converts selected text to uppercase. If no text is selected, it adds an uppercase version to the text in the text area.

Lowercase. Converts selected text to lowercase. If no text is selected, it adds an lowercase version to the text in the text area.
