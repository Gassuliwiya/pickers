This Unicode character app allows you to produce or analyse runs of Chinese text using the Simplified and Traditional Chinese orthographies. Character apps are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For release notes see the Github commit list.

About the app

There are a number of ways to find characters and add them to the large text box, including the following:

  • switch on Type-assist (on by default) and type QWERTY characters on your keyboard. As you type, the app shows alternatives in the native script that can be added to the text area by pressing a number key. The most common choice is usually added automatically, but will be overwritten if you choose a number key.
  • Latin type-assist does the same thing, but provides access to Latin characters used alongside the native text, or characters needed for phonetic or other transcriptions.
  • select characters from the area just below the large text box by clicking or tapping on them.

The app provides a range of other features that allow you to analyse or manipulate the content of the text box.

This character app doesn't provide an in-built IME, so you will need to use the one provided by your system. However, it does allow easy input of fullwidth alphanumerics, and punctuation, and there are ways to find Han characters. The app is particularly useful for analysing text, providing a way to get information about Han characters.

To find a Han character, type the pinyin equivalent, then use Lookup Han. This, along with a shortcut alternative approach, is described in more detail below. You can also look up hanzi using keywords in the definition, with the English to Han feature.

To get information about Han characters that are in the text area, use the Analyse text feature. While identifying most characters used for everyday Chinese, it produces pinyin transcriptions and definitions for around 3,000 common Han characters. These include the set indicated as main and auxiliary in CLDR 31.

Controls above the text area

Lookup Han. Allows you to look up Han characters with a given pronunciation.

Type the pronunciation in pinyin, using the Latin type-assist feature (hit the ~ key for a shortcut). Highlight the pinyin syllable you want to look up (only one syllable at a time), and click on Lookup Han. Matching hanzi characters will appear below the text area.

A shortcut after you have highlighted the pinyin syllable: hit the § key and the pinyin text will be replaced by the matching hanzi characters, surrounded by slashes (to keep them distinct from any other hanzi on the line).

English to Han. Allows you to look up Han characters by keywords from the definitions.

Type one or more words in English, highlight it, and select English to Han from the pulldown menu. Matching hanzi characters, with their definitions, will be displayed below the text area.

Toggle direction. Switches the writing mode of the text area between horizontal and vertical.
