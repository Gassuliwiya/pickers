List of pickers

Updated 31-Mar-2018 • tags apps, pickers, javanese

This Unicode character picker allows you to produce or analyse runs of Javanese text. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes characters in the Unicode Javanese block.

Also included in this picker is an SS shift button. If you click on this the consonant table is replaced by a table of subjoined/conjoined forms, in the same arrangement. This may be useful for spotting shapes. When you click on these, a prangkon is automatically inserted into the text area – there is no need to click on that separately.

Controls above the text area

Transliterate. Converts the contents of the text area to a Latin transliteration that provides a single correspondence between each Javanese symbol and a Latin symbol. It doesn't lead to a good phonetic transcription, but the process is reversible (unlike most phonetic transcriptions).

The transliteration scheme is one I made up, for lack of alternative standardised sources. Dots above and below consonants are used for medial and final forms, and dots below are used to distinguish independent vowels from vowel signs. An a is used to represent possible locations for the inherent vowel.

The transliteration algorithm is unable to detect where word boundaries occur, so it omits all spaces, and you'll need to introduce them manually after the transcription is done. It is also unable to detect whether ha with a vowel-sign, such as a, should be transcribed ha or a, and so it renders it always as ha. Finally, the punctuation of Javanese doesn't always map exactly to the right characters for a Latin transcription, so you may need to tweak that sometimes.
