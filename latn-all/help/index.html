This Unicode character app allows you to produce or analyse runs of Latin text. It is particularly useful for locating infrequently used characters, or characters that are spread across several Unicode blocks.

If something is broken or missing raise an issue. For version information see the Github commit list.

Inputting characters

This app includes characters in the Unicode Basic Latin, Latin-1 Supplement, Latin Extended A-E, and Latin Extended Additional blocks, as well as IPA and Phonetics blocks, and Modifier and Punctuation blocks.

To analyse text, copy & paste it into the large text area and select the desired control(s).

To produce text in the text area, click on one of the orange items in the selection area that resembles the shape of the character you are looking for. Then click on one of the grey shapes that appears below to add the character to the text area. The orange ellipsis (...) gives access to a mix of characters with shapes that didn't fit elsewhere.

To view uppercase characters, click on SHIFT just below the text area.

Uppercase. Converts selected text to uppercase. If no text is selected, it adds an uppercase version to the text in the text area.

Lowercase. Converts selected text to lowercase. If no text is selected, it adds an lowercase version to the text in the text area.
