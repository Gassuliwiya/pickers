Updated 28-Nov-2019 • tags apps, pickers, latin

This Unicode character app allows you to produce or analyse runs of Latin text. It is particularly useful for locating infrequently used characters, or characters that are spread across several Unicode blocks.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes characters in the Unicode Basic Latin, Latin-1 Supplement, Latin Extended A-E, and Latin Extended Additional blocks, as well as IPA and Phonetics blocks, and Modifier and Punctuation blocks.

To add a character in this character app, click on one of the orange items in the selection area that ressembles the shape of the character you are looking for. Then click on one of the grey shapes that appears below to add the character to the text area. The orange ellipsis (...) gives access to a mix of characters with shapes that didn't fit elsewhere.

To view uppercase characters, click on the Shift button just below the text area.

NOTE: Clicking on the shift button more than once may produce undesirable conversions – for example, under B there is ß: if you click on Shift it will uppercase to SS, and if you click the Shift button again it will then lowercase to ss. In this case, to get back to the original lowercase set of characters it is best to click on B again. Similar issues occur for the following characters: ǲ, ǅ, İ, ı, Ʃ, ẛ, ǈ, ǋ.

Controls above the text area

Uppercase. Converts selected text to uppercase. If no text is selected, it adds an uppercase version to the text in the text area.

Lowercase. Converts selected text to lowercase. If no text is selected, it adds an lowercase version to the text in the text area.
