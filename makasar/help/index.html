This Unicode character picker allows you to produce or analyse runs of text in the archaic Makasar script. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

Note: The only font I know of for Makasar is MakasarGraphite, so you will need to download the font for the picker to work correctly. Even then, because this is a Graphite font, it will only work on Firefox.

About the chart

Includes characters in the Unicode 11 Makasar block.

Fonts. To properly display the text you will need to use a Makasar font. A Makasar font is available from Anshuman Pandey. Note that the MakasarGraphite font is best used with Firefox, which supports Graphite.

Controls above the text area

Transliterate. Converts the contents of the text area to a Latin transliteration that provides a one-to-one correspondence between Makasar and Latin symbols.

The transliteration scheme was developed for this picker, and, where possible, uses symbols that represent sounds typically associated with their Makasar equivalents, rather than the sometimes fairly arbitrary selection of diacritics found in other systems. Of course, the complexities of mapping graphemes to sounds means that this is far from a real phonetic transcription, and it doesn't show inherent vowels, but the process is reversible (unlike most transcriptions).

Transcribe. Converts the contents of the text area to a Latin transcription. It is only a general approximation to the phonetics, and the process is not reversible. Note that this doesn't provide the unwritten syllable final ng or glottal stop, nor the geminated consonants, since it is simply a transcription of the written text. It does, however, handle angka and doubled vowel-signs by repeating the consonant of the previous syllable
