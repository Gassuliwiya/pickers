This Unicode character picker allows you to produce or analyse runs of Uighur text. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes characters in the Unicode Arabic block that are used for the Uighur language.

Controls above the text area

Transliterate. Converts the contents of the text area to a Latin transliteration that provides a single correspondence between each Uighur symbol and a Latin symbol. It doesn't lead to a good phonetic transcription, but the process is reversible (unlike most phonetic transcriptions).

Transcribe. Converts the contents of the text area to a Latin transcription based on the Uyghur Latin alphabet. This is not easily reversible.

Input aids

ULY to Uighur. Allows you to generate Uighur text from a transcription based on the Uyghur Latin alphabet. It works as described in the section just above about Reverse transliteration, but uses a different panel to set up the transcription type-in mode.
