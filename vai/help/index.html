List of pickers

Updated 26-Nov-2018 • tags apps, pickers, vai

This Unicode character picker allows you to produce or analyse runs of Vai text. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

If something is broken or missing raise an issue. For version information see the Github commit list.

About the chart

Includes all the characters in the Unicode Vai block.

There are two ways of accessing the characters: (a) through a 2-click selector based on phonetics, and (b) from a table. The former is for people who know the phonetics and characters well. The latter is for people who are unfamiliar with the script.

To switch between the two, click on TABLE VIEW.

Notes:

  • the shape lookup feature is designed to work with the table only.
  • the table view doesn't allow selection or shape lookup of historical syllables, numbers, or logograms – for those, switch to the 2-click input.
  • the table view generally follows a standard layout, but is made more compact than normal in order to aid input. Nasalised syllables don't occupy a line of their own (except for the 2nd row), but instead appear alongside the unnasalised syllable (eg. ꗇꗈ). Also, the commonly used logograms and syllable-final letters are squeezed into some otherwise empty space near the top of the table (look for the coloured background).

Controls above the text area

Transliterate. Converts the contents of the text area to a Latin transliteration that provides a unique correspondence between each Vai symbol and a Latin symbol. It doesn't lead to a good phonetic transcription, but the process is reversible (unlike most phonetic transcriptions).

Reverse translit. Converts a Latin transliteration to Vai text. Pickers usually include the reverse transliteration command in the bar on the left, but the size of Vai makes the normal approach unfeasible. Instead, just write the text in Latin in the main box, highlight it, and click the Reverse translit. button. You can then quickly pull the Vai text into the main box using the controls to the right.
